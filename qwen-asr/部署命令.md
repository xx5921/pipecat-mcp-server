# Qwen3-ASR 部署命令

```bash
MSYS_NO_PATHCONV=1 docker run -d \
  --name qwen3-asr \
  --gpus all \
  --shm-size 4g \
  -p 8200:8200 \
  -v "E:\\qwen-asr\\workplace:/data/shared/Qwen3-ASR" \
  qwenllm/qwen3-asr:latest \
  qwen-asr-serve /data/shared/Qwen3-ASR/Qwen3-ASR-1.7B \
    --gpu-memory-utilization 0.4 \
    --max-model-len 8192 \
    --enforce-eager \
    --served-model-name qwen3-asr \
    --host 0.0.0.0 \
    --port 8200
```

- **模型名**：`qwen3-asr`
- **端口**：8200
- **显存**：约 8 GiB
- **模型文件**：`E:\qwen-asr\workplace\Qwen3-ASR-1.7B\`
